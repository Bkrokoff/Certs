Default Gateway

A default gateway is a device that forwards data from one network to another
A majority of the time this is going to be a router

A default gateway lets devices from one network communicate with devices on another network
"Default" means that the designated device is the first option that's looked upon when data needs to exit the network

If the computers on the left-hand side wanted to communicate with each other, they would communicate through the switch
They can do this because they are all on the same network - The data does not have to exit the network and go through the default gateway

How do computers know if another computer is on their own network or on a different network?

This involves IP addresses and subnets

192.168.0.2
An IP address consists of two parts - A network address and a host address

255.255.255.0 A subnet mask reveals how many bits in the IP address are used for the network by masking the network portion of the IP address

Since the computers are on different networks - the computer will send out an ARP broadcast and ask for the MAC address of the default gateway and not the mac address of the computer
