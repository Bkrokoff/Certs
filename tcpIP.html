TCP (Transmission Control Protocol) IP (Internet Protocol) Model

A model designed to standardise computer networking
Specifies how devices exchange information over the internet to one another
Protocols identify how data is broken down, addressed, transmitted, routed, and received

TCP
- Defines how applications create communication channels
- Manages how a message is broken down to be transmitted

IP
- Defines how to address and route packets for delivery

Layers of TCP/IP model

Application Layer

The application layer provides applications with standardized data exchange
Example: HTTP, FTP, Post Office Protocol 3, Simple Mail Transfer Protocol, Simple Network Management Protocol

Transport Layer

Responsible for maintaining end-to-end communications across the network

Network Layer

Data Link

Physical Layer

Special Notes: - Similar to OSI Model
