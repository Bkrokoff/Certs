Storage Devices

A computer has two types of memory - Primary and Secondary
Secondary memory is permanent which is the hard disk drive

Hard Drive (Magnetic)

A sealed case that contains magnetic disks where the data is stored on
The disk rotates at high speeds and the actuator either writes data to the disk or reads data from them (Looks similar to record player)
The disk rotates at either 5400 - 7200 - 10000 RPM
Typical desktop hard drive would operate at 7200 RPM
Cheaper hard srives would operate at 5400 RPM - typical for laptops
Hard drives are non volatile because they can retain and store data when not connected to a power source

SATA Hard Drive

Hard drives today uses the Serial ATA interface
A newer standard thats faster than the old Parallel ATA
Instead of data travelling in a parallel path, it now travels in a serial path with a SATA cable (Travels 1 bit at a time)
SATA - Transfer speeds average 6 GB/s
Hot swappable and smaller cables

Solid State Drives (SSD)

Has no moving parts and uses flash memory chips to store data
Since it has no moving parts, data transfer is very fast

Hybrid Drive (Magnetic and SSD)

Combines the use of magnetic disks and flash memory
Takes advantage of the large capacity and low cost of magnetic disks, with the speed of an SSD
Magnetic disks will be used to store data, while the flash memory would be used for the caching of the data
Used when cost is an issue, because SSD Drives are more expensive than magnetic disks

Removeable Storage

Tape Drives

An inexpensive way to back up data from your hard drive
It attaches to a computer, usually with a USB cable
Data is backed up sequentially to tape
To restore data, you have to restore data in the same order as you backed them up

CD-ROM Drives

Has become a standard component in computers today
Reads data from a CD-ROM which is a disk that stores permanent data
CD-Rom stands for compact disk read only memory, which means that the data can only be read and cannot be changed or written to

CD-RW Drive

CD-RW drive can change or write data to a CD
Has a moving laser assembly that writes or "burns" the data onto the CD
CD-R: can only be written once
CD-RW: allows data to be rewritten multiple times
A CD can hold 700 MB of data

DVD-ROM Drive

DVD-ROM drive is a device that reads DVD disks
Has largely replaced ordinary CD-ROM drives beacuse of large storage capacity of DVD disks
DVD can store approx. 4.7 GB of data

DVD-RW Drive

DVD-RW drive can change or write data to a DVD
Writes data on to DVD-R disks and rewrites data onto DVD-RW disks
Dual layer DVD-RW drives can burn dual layer DVDs
Dual layer DVDs have 2 recordable layers
Dual layer DVDs can hold approx 8.5 GB of data

Blu-Ray Drive

The latest optical drive to date that released in 2006
Will eventually replace the DVD format because of its high capacity
Blu-Ray disc can hold 25 GB of data, which is 5 times more than a DVD

BD-R & BD-RE Drive (Blu-Ray)

BD-R drive burns data to a blu-ray disc one time
BD-RE drive can erase data and overwite data on a blu ray disc
Dual layer BD-Rs can hold 50 GB of data

Thumb Drive

Uses memory chips to store data
Miniature storage device with a USB interface
Can store anywhere from 500 MB to 256 GB
Fast and convenitent data storage

SD Card

Secure digital cards are a popular type of flash memory
Primarily used in digital cameras for storing photographs and videos
Come in various capacities, anywhere from 2 GB - 512 GB
Need an SD card reader to be read on the computer

Storage Cards

Other types of SD cards include the mini SD and the micro SD
The micro SD card is what is commonly used in most smart phones for adding extra storage
Another type of flash drive used in a lot of DSLR cameras is compact flash
Provides quick and easy storage for pictures and videos in DSLR cameras
A type of non-standard SD card would be the older XD (extreme digital) picture card
Proprietary in Olympus and Fujifilm cameras

eMMC Memory

This kind of memory is what you would typically find inside smart phones and tablets

Hot Spares and Cold Spares

Hot - Equipment that can be swapped out without the need of turning off the power
Cold - Power must be turned off before replacing the piece of equipment
