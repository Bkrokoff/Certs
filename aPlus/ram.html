RAM (Random Access Memory)

RAM is primary memory or temporary storage
In order for a program to run it needs to be loaded into RAM first
Increasing RAM will make the computer run faster

Requires constant electrical power to store the data
RAM is stored on the motherboard in modules called DIMMs
DIMMs come in different memory sizes
Range anywhere from 128 MB to 64 GB per DIMM

Ram Types

DRAM - Dynamic RAM

Contains capacitors
Has to be refreshed often

SRAM - Static RAM

Uses transistors
Does NOT have to be refreshed
Faster than DRAM
Very expensive
Examples: Memory cache used by the CPU

SDRAM - Synchronous Dynamic RAM

Used today in RAM DIMMs
Difference between SDRAM vs DRAM:
DRAM - Operates Asynchronously with the system clock
SDRAM - Operates synchronously with the system clock
All signals are tied to the system clock for a better controlled timing

RAM Speed

PC-100

100 Mhz = The speed at which it operates
8 byte wide bus
Total bandwidth = 100 Mhz * 8 bytes = 800 Mb/s

PC-133

133 Mhz = The speed at which it operates
8 byte wide bus
Total bandwidth = 133 Mhz * 8 bytes = 1066 Mb/s

DDR - Double Data Rate

Sends double the amount of data in each clock signal
DDR uses the total bandwidth in its label name

PC - 2700

333 Mhz clock speed
8 byte wide bus
Total bandwidth = 333 Mhz * 8 = 2700 Mb/s

PC - 3200

400 Mhz clock speed
8 byte wide bus
Total bandwidth = 400 Mhz * 8 = 3200 Mb/s

DDR2

A newer technology that has succeeded DDR
Faster than DDR
Allows for higher bus speeds
Uses less power than DDR
Has 240 pins vs 180 pins (DDR)

DDR3

Twice as fast as DDR2
Bandwidths of over 12800 Mb/s
Has 240 pins
Bottom notches are in different places
Not interchangeable with DDR or DDR2
DDR3 Examples: PC3 - 8500, PC3 - 12800

RDRAM (Rambus Dynamic Ram)

RIMM - Rambus Inline Memory Module

Has 184 pins
Look similar to DIMMs
Bottom notches are in the center of the module
Debuted in 1999
Was a breakthrough in the speed of memory
Has quickly fallen behind due to advancment of technology in DIMMs

RDRAM speed was at 800 Mhz (SDRAM speed was 133 Mhz)
Only had a 2 byte wide bus (SDRAM has an 8 byte wide bus)
800 Mhz * 2 bytes = 1600 Mb/s

RIMM technology was designed to work with a continuous signal
All the memory slots on the motherboard must be used to work properly
C-RIMM: Continuity RIMM (dummy RIMM) can be used instead of RIMMS

Dual Channel Mode

Requires a pair of identical DIMMs installed on the motherboard
Allows the memory controller the ability to communicate with 2 DIMMs simultaneously
Increase the speed of accessing the memory

Motherboard must be equipped to work in dual channel mode
DIMMs must be identical to each other in speed, size, and features
DIMMs must be inserted into the motherboard in a specific slot configuration

Triple Channel Mode

Not very common
Allows the memory controller the ability to communicate with 3 DIMMs at the same time
Modern motherboards have a 64 bit architecture
Single channel: Transfers 64 bits of data at a time
Dual channel: Transfers 128 bits of data at a time
Triple channel: Transfers 192 bits of data at a time
DIMMS have to be inserted into the same color slots and must be identical in order for them to work properly

Single Sided vs Double Sided

Doesn't necessarily refer to the physical location of the memory chips that are on the sides of the memory module
It refers to the groups of memory chips that a memory controller accesses
For example: Double sided RAM has two groups of memory chips
Double sided memory means that the memory controller sees these two groups of memory chips separately

Single sided RAM has one group of memory chips
With single sided RAM, the memory controller can access it as one group
Which makes single sided RAM faster than double sided RAM

ECC Memory (Error Correcting Code)

Detects if the data was correctly processed by the memory module
Makes a correction if it needs to
Non ECC - 8 memory chips
ECC - 9 memory chips
Most RAM modules today are non-ECC
ECC memory is mostly used in servers

Buffered vs Unbuffered RAM

Buffered RAM (registered RAM) is made to add stability to RAM
Adds an extra register between the RAM and the memory controller
Stores (buffers) data before it gets sent to the CPU
Adds stability and reliability in computer systems that have a lot of memory installed
Used mainly in servers

SODIMM (Small Outline Dual Inline Memory Module)

RAM that is used in smaller devices, such as laptops
Comes in different types, suchas DDR, DDR2, DDR3
