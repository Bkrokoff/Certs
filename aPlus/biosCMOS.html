BIOS / CMOS

BIOS (Basic Input Output System)

The BIOS is software that is built into the motherboard that initializes the computers peripherals
Such as the keyboard, mouse, video card, optical drives, etc
After initialization, it then searches for a boot device (OS)

The BIOS software is stored on the BIOS chip on the motherboard
The BIOS chip is non-volatile
Popular vendors of BIOS chips are American Megatrends and Pheonix Technologies

CMOS Battery

Computers need to maintain certain settings (date & time, hardware settings) that are stored on the CMOS chip
The CMOS (Complimentary Metal Oxide Semiconductor) battery maintains these settings
A small button cell battery located on the motherboard

Boot Priority

When you turn on a computer, the computer does a POST - Power On Self Test - to ensure that everything is working
After POST has passed, the BIOS searches for something to boot from (OS)
You can change the boot priority in CMOS in any order you want
After you make the change, you must save the settings and the settings will take affect after the computer is rebooted

BIOS

You can set the date and time in BIOS
You can also enable or disable certain devices in the BIOS
Allows you to modify the CPU clock speed
You can monitor the temperature of the CPU with the BIOS
Virtualization can be enabled or disabled in the BIOS
A BIOS password protects against unauthorized users
Prevents changes to the BIOS and from the computer booting

POST (Power on Self Test)

The POST tests the computer to make sure all the requirements are met and the hardware is working correctly
If the computer passes the test, it will make a short single "beep"
No beep or multiple beeps, indicates a problem
The computer will generate a beep code to help you pinpoint the problem
Refer to the manufacturers documentation to find a list of beep codes

BIOS

Some BIOS systems detect if the computer case has been opened

LoJack - Service that can locate your computer if it has been stolen

UEFI (Unified Extensible Firmware Interface)

UEFI (NEW)
More modern BIOS
User friendly, graphical user interface
Recognizes larger storage devices
Able to use a mouse in the interface

BIOS (OLD)
Blue screen that resembles the BSOD (Blue Screen of Death)
Cannot recognize larger storage drives
Only able to use a keyboard

Secure Boot - built in new feature in UEFI that stops any digitally unsigned drivers from loading and helps to stop malicious software such as rootkits
